getFields ( stdClass $  params)

We can get the form fields by form name or form if example: getFields('Demographics') or getFields('1') The logic of the function is to get the form parent field and its options, then get the child items if any with it options. Then.. use reg Expression to remove the double quotes from all the options and leave the double quotes to all options values, unless the value is a int or bool.

[{ "xtype":"fieldset", "title":"Who", "collapsible":"true", "items":[{ "xtype":"fieldcontainer", "fieldLabel":"Name", "layout":"hbox", "anchor":"100%", }] }]
