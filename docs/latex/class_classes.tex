\subsection{\-Detailed \-Description}
\-Intermediate class parsing structure.

\-The \hyperlink{classphp_documentor___intermediate_parser}{php\-Documentor\-\_\-\-Intermediate\-Parser} class uses this class and its cousin, \hyperlink{class_procedural_pages}{\-Procedural\-Pages} to organize all parsed source code elements. \-Data is fed to each immediately after it is parsed, and at conversion time, everything is organized.

\-The \hyperlink{class_classes}{\-Classes} class is responsible for all inheritance, including resolving name conflicts between classes, determining which classes extend other classes, and is responsible for all inheritance of documentation. \{
